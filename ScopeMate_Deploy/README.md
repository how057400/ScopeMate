# ScopeMate v1.6

웹 프로젝트 예상 공수와 권장 일정을 계산하는 MVP 웹앱입니다.

## 실행 방법

`index.html`을 크롬으로 열면 실행됩니다.

## 계산 방식

기본 공수
+ 반응형 추가 공수
+ 디자인 시스템 추가 공수
+ 인터랙션 추가 공수
+ CMS 추가 공수
+ 개발 가이드 추가 공수
= 기준 공수

## v1.6 기준

실무에서 질문으로 드러나지 않는 작업을 기본 공수에 포함했습니다.

포함된 숨은 작업:
- 레퍼런스 탐색
- 방향성 검토
- 내부 피드백 반영
- 여러 시안 테스트
- 기본 정리 및 커뮤니케이션
- 작업 중간 검토 및 보정

대표 기준:
- 단일 페이지: 32h
- 웹사이트 구축: 56h
- 기타: 40h

결과 범위는 리스크가 높을수록 더 넓게 표시됩니다.
